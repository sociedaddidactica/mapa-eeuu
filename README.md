# Mapa interactivo de Estados Unidos

Este proyecto muestra un ejemplo sencillo de un mapa de los 50 estados de EE. UU. Cada estado puede pulsarse para desplegar algunas ciudades y acceder a las páginas con la forma `https://855anyjunk.com/[ciudad]`.

El mapa SVG empleado en `assets/us-map.svg` es un mock simplificado dispuesto en una cuadrícula para propósitos demostrativos.

## Estructura del proyecto

- `index.html` – Página principal que carga el SVG y la lista de ciudades.
- `css/styles.css` – Estilos básicos.
- `js/app.js` – Lógica de interacción y datos de ciudades.
- `assets/us-map.svg` – Representación de los estados.

Para probar el proyecto basta con abrir `index.html` en el navegador.
